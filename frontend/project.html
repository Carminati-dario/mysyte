This are some of my projects

CLEAR

Contact Lenses Engagement And Renewal.
CLEAR is a project born for the management of the 365LAC contracts. The main goal was to automate a process that was previously done manually, drastically reducing management time and the risk of human errors.
Developed entirely by me in 2025, the application is a web-based solution.
It works by a web application. The technology used is Go for the backend and JavaScript for the frontend and styled with CSS.

The Prison Escape

The Prison Escape is a Korean logic puzzle. The game consists of a grid in which you must move blocks of different sizes to clear the path and let the red block escape.
The game was implemented using HTML for structure, JavaScript for interactivity, CSS for styling, and React for building the user interface.
The repository is available on my GitHub.

GoGroup

GoGroup is a school project focused on the implementation of a TCP-based multi-chat system using Go.
The main goal was to design a chat system capable of handling multiple groups, where each user can join a specific group and communicate only with its members.
The system is composed of three main components: Server, which manages communication, groups, and message routing between users; Client, which provides a user interface allowing users to send messages, choose or create groups, and set a username; and Screen, which displays messages received from the server.
The repository is available on my GitHub.
